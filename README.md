# 数据结构学习与资料仓库

本仓库汇总了数据结构课程的代码示例、练习题解、课件与历年试卷等资料，涵盖顺序表、链表、栈与队列、树、图、排序、查找、指针与动态内存等核心主题，同时包含算法竞赛与课程设计相关资源。

## 仓库结构

- `Improvement/`：课堂巩固练习与进阶示例（链表基础、链表应用、二叉树遍历等）。
- `孙逊老师数据结构课（2024）/`
  - `代码/`：按专题拆分的 C 语言示例与练习（函数、动态内存分配、数组、字符串、结构体、线性表、栈与队列、树、图、排序、查找、算法分析等），多数同名无后缀文件为可执行文件或输入样例。
  - `课件/`：课程 PPT/PDF（数据结构基础、算法分析、线性表、栈与队列、树、图等）。
- `往年资料/`：历年试卷与参考答案（PDF、DOCX）。
- `数据结构课程设计/`：课程设计资料与素材（大量图片、音频等资源）。
- `第八届传智杯/`：竞赛题目的 C++/Python/JavaScript 参考解答。
- `算法新生赛/`：算法新生赛题目解答（JavaScript 等）。
- `解答小登问题/`：若干 HTML 示例与练习素材。
- `.vscode/`：VS Code 编译与调试配置（含 C/C++ 相关 tasks/launch）。
- `.cursorrules`：团队约定（中文交互与注释、提交规范等）。

> 仓库体量较大，`数据结构课程设计/` 与部分代码目录含大量资源文件，克隆或复制时请预留足够磁盘空间。

## 运行与编译

- C/C++ 示例
  - 推荐编译器：`gcc`/`clang`（类 Unix 环境）或 `MSVC`/`mingw64`（Windows）。
  - 单文件示例可直接编译：`gcc test1.c -o test1`，再执行 `./test1`（Windows 为 `test1.exe`）。
  - 若目录内存在同名可执行文件（无扩展名），可直接运行；如需重新编译，请覆盖生成的可执行文件。
- Python/JavaScript 示例
  - Python：`python 文件名.py`
  - Node.js：`node 文件名.js`
- 若需调试，`.vscode/launch.json` 和 `.vscode/tasks.json` 已提供基础模板，可按需调整编译参数与工作目录。

## 学习路径建议

1. 先通读 `课件/`（数据结构基础 → 线性表 → 栈与队列 → 树 → 图 → 排序/查找）。
2. 按专题运行 `代码/` 中对应示例，理解接口与时间复杂度。
3. 参考 `Improvement/` 和竞赛目录的题解，练习实际问题建模与实现。
4. 使用 `往年资料/` 进行自测，查漏补缺。
